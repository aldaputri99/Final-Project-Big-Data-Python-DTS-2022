==Library Python yang harus di-install==
pip install flask_mysqldb
pip install plotly
pip install sqlalchemy

==Setting Database==
1. Download XAMPP; Install
2. Run XAMPP; Start Apache & MySQL
3. Buka "localhost/phpmyadmin" di browser
4. Klik "New" di kiri atas; Database name: db_quithon
5. Di db_quithon, Klik 'Import' di barisan menu atas
6. Choose file yang ada di folder db: db_quithon.sql; Scroll ke bawah, klik Import

==Buka web==
1. Buka folder "web_trial_2" di VS Code
2. Buka file main.py; Run
3. Buka "localhost:5000" di browser
4. Klik Masuk/Daftar;
